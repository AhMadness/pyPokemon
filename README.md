# Pokemon Type Matchup Studio (Pre-Fairy)

A modern Python desktop app for exploring **Pokemon type matchups** using the official **pre-Fairy (17-type)** chart.

## Features

- Main matchup view with fixed-size cards
- Offensive + defensive tabs
- `x4` indicator on type chips when applicable
